# tutedude-assignment4

$ Task 1: Reading a File Line by Line
Description

This task focuses on:

Opening and reading a text file named sample.txt

Printing its contents line by line

Handling the error if the file does not exist

Features

Uses try-except to catch FileNotFoundError

Ensures the program does not crash if the file is missing

Expected Output

If the file exists, its contents are printed line by line.

If the file does not exist, an error message is displayed.

$ Task 2: Writing, Appending, and Reading a File
Description

This task demonstrates:

Taking user input and writing it to a file named output.txt

Appending additional data to the same file

Reading and displaying the final content of the file

Features

Uses write ("w"), append ("a"), and read ("r") modes

Preserves existing data while appending

Displays the complete file content after modifications
